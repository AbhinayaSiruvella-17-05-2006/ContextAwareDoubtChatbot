📚 Context-Aware Doubt Chatbot

🚀 Overview

The Context-Aware Doubt Chatbot is an intelligent learning assistant that helps students resolve doubts instantly while studying.

Instead of switching platforms or searching manually, users can ask questions directly within their study material and get relevant, context-based answers.

---

🎯 Problem Statement

Students often face doubts while studying from notes or digital content.
However:

They cannot resolve doubts instantly

They lose context when switching platforms

Answers found online are often generic

👉 This breaks learning flow and reduces understanding.

---

💡 Solution

We built a context-aware chatbot that:

Works directly inside learning content

Understands the lesson context

Provides instant and relevant answers

This ensures a smooth and uninterrupted learning experience.

---

✨ Features

📖 Content Handling

View online learning material

Upload files or add text content

Display lesson content in-app

🤖 Chatbot Interaction

Ask questions directly

Highlight specific text and ask doubts

Context-based responses (not generic)

🎤 Voice Support

Voice input (ask questions)

Voice output (listen to answers)

🌍 Language Translation

Supports multiple languages

Translates entire app content

🧾 Chat Management

Clear chat option

Store chat history

Download chat history

---

🧠 Core Concept: Context Awareness

The chatbot understands:

Selected text

Current section

Entire lesson

👉 And generates answers based only on that context.

---

🔄 System Flow

User opens the app

Chooses:

View Material OR

Post Material

Inputs content (text/upload)

Content is displayed

User asks question / highlights text / uses voice

Context is extracted

Backend processes request

LLM generates response

Response shown (text + voice)

Chat is stored and can be downloaded
